# Java multi room chatting Server

자바 채팅방 채팅 서버( 콘솔 ).

Java multi chatting server( console ).

## Dev Environment

- Language : java 8
- java Thread



## Introduce Program

- Server save accepted Client to ```HashMap<String, List<Socket>>```
- Client Socket add List<Socket> and put Map with Chatting room name.
